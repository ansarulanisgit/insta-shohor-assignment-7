Javascript is a dynamic typed object oriented programing language and it is an interpreted language too. It runs within browsers throung Javascript engine. There are several popular Javascript engine like V8 in Google Chrome, Spidermonkey in Firefox etc. Javascript engine runs the Javascript codes in real time. When we run Javascript codes, at first JS engine send the codes to Parser. The Parser checks the codes for errors and validate it. After that it creates a data sturcture called AST(Abstract Syntax Tree). Then JS engine converts the AST to byte size machine code. After that it send the converted machine codes to system to execution.

We can access browser storage properties using Web Storage API. It provides access local storage or sessions storage of a particular domain. We can store, modify and delect data using it. The localStorage and sessionStorage are almost similar. They store data by key/values pair of a particular domain in string format. The major difference between those is the expiration date. sessionStorage data is removed when the particular browser tab or window is closed. But localStorage data persists even domain tab, window or browser is closed. We can get the localStorage data of the domain again when reopen the browser and site.
